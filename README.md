
# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior to uncover spending patterns, customer segments, and revenue drivers.

## Objectives
- Identify high-value customers
- Understand demographic impact on spending
- Segment customers using clustering
- Build predictive model for purchase amount

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
- Streamlit / Power BI

## Key Insights
- 26-35 age group contributes highest revenue
- High-spending cluster identified via KMeans
- Purchase amount positively correlated with age (moderate)

## How to Run
1. Clone repo
2. Install requirements
3. Run notebook or streamlit app
